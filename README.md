# AI-IoT-Kunskapskrav-1

Repository för min inlämningsuppgift i kursen AI och IoT.

Här finns samtliga filer och dokument kopplade till Kunskapskontroll 1. Arbetet har genomförts iterativt, vilket innebär att utvecklingen skett stegvis med tester och förbättringar längs vägen.

För att göra det enklare att följa projektet rekommenderas följande läsordning:

## 📌 Rekommenderad läsordning

1. **RAG-chattbot – projektbeskrivning och slutsats**  
   Översikt av problem, lösning, resultat och reflektion.

2. **Exploring/**  
   Inledande analyser och utforskning av data och metod.

3. **Chunking_och_embeddings_2.ipynb**  
   Utveckling av chunking-strategi och embeddings  
   - `embedding_service.py`  
   - `vector_store.py`

4. **Retrieval_test_final.ipynb**  
   Tester av retrieval och likhetsberäkning.

5. **RAG_model_mer_tester.ipynb**  
   Vidare experiment med modell och prompt.

6. **evaluering_final.ipynb**  
   Evaluering av modellens prestanda.

7. **RAG_modell_final.ipynb**  
   Slutlig implementation av RAG-chattboten.
